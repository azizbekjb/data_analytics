# Data Analytics

Bu mening birinchi Ma'lumotlar tahlili bo'yicha o'rgangan bilimlarimni bayon qiladi.

Ushbu repoda Pandas, Numpy va Matplotlib kutubxonalari yordamida fayllar tozalangan, tahrirlangan va tahlil qilingan.

## Repo nimalarni o'rgatadi?
- Numpy, pandas va matlotlib kutubxonalari asosalari.
- array, series, dataframe haqida tushunchalar.
- Fayllarni tahlil qilish.
- Sodda grafiklar chizish.

## Repodan foydalanish bo'yicha qo'llanma.

1. Reponi klonlang. Terminalni oching va quyidagi kodni yozing.
```bash
git clone https://github.com/azizbekjb/data_analytics.git
```

2. Reponi o'zingizga qulay bo'lgan IDE(Dasturlash muhiti)da oching.

3. IDE ichida terminalni oching va quyidagi kodlarni navbat bilan yozing.
    - Virtual muhit yarating.
    ```bash
        python3 -m venv venv
    ```
    - Virtual muhitni faollashtiring.
    ```bash
        source venv/bin/activate
    ```

    - Kerakli kutubxonalarni o'rnating.
    ```bash
    pip install -r requirements.txt
    ```

4. Har bir faylni ishlatib natijalarni oling.
